# Superstore Sales & Profit Dashboard

## Project Overview
This project analyzes Superstore sales and profit performance using Python and Power BI.

## Objective
To clean and analyze sales data and build an interactive dashboard to identify sales trends, profit patterns, and business performance.

## Tools Used
- Python
- Pandas
- Power BI
- CSV Dataset

## Data Preparation
- Cleaned and processed raw CSV data using Python
- Handled missing values and prepared dataset for visualization

## Dashboard KPIs
- Total Sales
- Total Profit
- Category Performance
- Regional Sales Analysis

## Key Insights
- Identified top-performing categories
- Compared regional sales and profit
- Analyzed overall business performance
